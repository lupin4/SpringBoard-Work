Selecting Elements in the DOM

There are several methods to select elements in the DOM:

  • getElementById('id')
  • getElementsByClassName('class')
  • getElementsByTagName('tag')
  • querySelector('selector')
  • querySelectorAll('selector')
  1. getElementById selects an element with a specific id. It returns a single element.
  2. getElementsByClassName selects all elements with a specific class. It returns a collection of elements.
  3. getElementsByTagName selects all elements with a specific tag name. It returns a collection of elements.
  4. querySelector selects the first element with a specific CSS selector. It returns a single element.
  5. querySelectorAll selects all elements with a specific CSS selector. It returns a collection of elements.
