Array Methods

forEach()

  • The forEach() method executes a provided function once for each array element.
  • The forEach() method does not return anything.
  • The forEach() method does not change the original array.
  • The forEach() method is used to iterate over an array.
  • The forEach() method is used to perform a task on each element of an array.

map()

  • The map() method creates a new array populated with the results of calling a provided function on every element in the calling array.
  • The map() method returns a new array with the results.
  • The map() method does not change the original array.
  • The map() method is used to transform data.
  • The map() method is chainable.
  • The map() method is used to create a new array from an existing array.

filter()

  • The filter() method creates a new array with all elements that pass the test implemented by the provided function.
  • The filter() method returns a new array with the elements that pass the test.
  • The filter() method does not change the original array.
  • The filter() method is used to filter data.
  • The filter() method is chainable.
  • The filter() method is used to create a new array from an existing array.

some() and every()

  • The some() method tests whether at least one element in the array passes the test implemented by the provided function.
  • The some() method returns a boolean value.
  • The some() method does not change the original array.
  • The some() method is used to check if at least one element in the array satisfies a condition.
  • The some() method is chainable.
  • The some() method is used to create a new array from an existing array.

find() and findIndex()

  • The find() method returns the value of the first element in the array that satisfies the provided testing function.
  • The find() method returns a single value.
  • The find() method does not change the original array.
  • The find() method is used to find the first element in the array that satisfies a condition.
  • The find() method is chainable.
  • The find() method is used to create a new array from an existing array.
  • The findIndex() method returns the index of the first element in the array that satisfies the provided testing function.
  • The findIndex() method returns a single value.
  • The findIndex() method does not change the original array.
  • The findIndex() method is used to find the index of the first element in the array that satisfies a condition.
  • The findIndex() method is chainable.
  • The findIndex() method is used to create a new array from an existing array.

reduce()

  • The reduce() method executes a reducer function on each element of the array, resulting in a single output value.
  • The reduce() method returns a single value.
  • The reduce() method does not change the original array.
  • The reduce() method is used to reduce the array to a single value.
  • The reduce() method is chainable.
  • The reduce() method is used to create a new array from an existing array.

includes()

  • The includes() method determines whether an array includes a certain value among its entries, returning true or false as appropriate.
  • The includes() method returns a boolean value.
  • The includes() method does not change the original array.
  • The includes() method is used to check if an array includes a certain value.
