Array Methods

forEach()

  • The forEach() method executes a provided function once for each array element.
  • The forEach() method does not return anything.
  • The forEach() method does not change the original array.
  • The forEach() method is used to iterate over an array.
  • The forEach() method is used to perform a task on each element of an array.

map()

  • The map() method creates a new array populated with the results of calling a provided function on every element in the calling array.
  • The map() method returns a new array with the results.
  • The map() method does not change the original array.
  • The map() method is used to transform data.
  • The map() method is chainable.
  • The map() method is used to create a new array from an existing array.

filter()

  • The filter() method creates a new array with all elements that pass the test implemented by the provided function.
  • The filter() method returns a new array with the elements that pass the test.
  • The filter() method does not change the original array.
  • The filter() method is used to filter data.
  • The filter() method is chainable.
  • The filter() method is used to create a new array from an existing array.
